Course Curriculum - Python 2

Welcome to the Advanced Python Programming course! This 8-week course is designed to deepen your understanding of Python and introduce more complex concepts. Ideal for students who have completed Python 1 or have equivalent experience.

Week 1: Advanced Data Structures

    Lesson 1: Sets and Tuples
    Lesson 2: Stacks and Queues
    Project 1: Implementing a Stack and Queue

Week 2: Functional Programming

    Lesson 3: Lambda Functions
    Lesson 4: Map, Filter, and Reduce
    Project 2: Data Analysis with Functional Programming

Week 3: File Handling and Context Managers

    Lesson 5: Reading and Writing Files
    Lesson 6: Using Context Managers
    Project 3: File-Based Data Processing Tool

Week 4: Object-Oriented Programming in Depth

    Lesson 7: Class Inheritance
    Lesson 8: Magic Methods and Operator Overloading
    Project 4: Building a Custom Class Hierarchy

Week 5: Modules and Packages

    Lesson 9: Creating and Using Modules
    Lesson 10: Packaging Python Projects
    Project 5: Creating a Python Package

Week 6: Error Handling and Logging

    Lesson 11: Exception Handling Techniques
    Lesson 12: Logging in Python
    Project 6: Error Logging System for a Python Application

Week 7: Introduction to Web Development with Flask

    Lesson 13: Setting Up Flask
    Lesson 14: Building Your First Web App
    Project 7: Simple Flask Web Application

Week 8: Final Project

    Lesson 15: Project Planning and Requirements
    Lesson 16: Final Project Presentation
    Project 8: Choose Your Own Advanced Project (Web App, Data Analysis Tool, etc.)
